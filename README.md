# 🏢 Enterprise IT Infrastructure Lab

A hands-on Windows Server project that simulates a real enterprise IT environment. The purpose of this project is to learn, practice, and document the day-to-day responsibilities of a Windows System Administrator.

## 🎯 Project Goal

The goal of this project is to build a complete Windows-based enterprise infrastructure from scratch while following real-world IT administration practices. Every phase is documented step by step to demonstrate both technical skills and professional documentation.

## 🛠️ What This Project Covers

* Windows Server 2022 Installation
* Active Directory Domain Services (AD DS)
* DNS Configuration
* DHCP Configuration
* Organizational Units (OUs)
* User & Group Management
* Group Policy (GPO)
* File Server & NTFS Permissions
* PowerShell Automation
* Security Best Practices
* Troubleshooting
* IT Documentation

## 📚 Project Structure

Each phase includes:

* **Objective** – What we are building
* **Why it matters** – Its role in an enterprise environment
* **Step-by-step implementation** – Clear and easy-to-follow instructions
* **Verification** – How to confirm the configuration works
* **Screenshots** – Visual proof of the completed work
* **Key takeaways** – What was learned in the phase
